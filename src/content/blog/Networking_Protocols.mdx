# Networking Protocols

Networking protocols are the agreed rules and conventions that enable devices to communicate reliably across networks. They define message formats, timing, error handling, and procedures for connection management. Understanding key protocols helps design, troubleshoot, and secure modern networked systems.

## Conceptual Models

- OSI and TCP/IP models
- OSI model divides networking into seven layers from physical to application.
- TCP/IP model is a practical four layer stack used on the internet: link, internet, transport, application.
